## Overview

Hmovie is a comprehensive digital catalog and film database focused on preserving the history of Hmong cinema, filmmaking, and storytelling. This static site contains over 300 entries and offers a searchable, browsable collection that documents the rich cinematic traditions of the Hmong community.

Built with Jekyll and structured using simple Markdown files for each movie entry, the project makes it easy for contributors to add new titles, update existing information, and help maintain this cultural archive. Entirely community-managed, Hmovie welcomes anyone interested in preserving Hmong film heritage.

## Getting Started

Clone the repository to your local machine:

```bash
git clone https://github.com/brandonmoore1994/hmovie-community-hub.git
cd hmovie-catalog
```

To run the site locally, ensure Jekyll is installed, then:

```bash
bundle install
bundle exec jekyll serve
```

The site will be available at `http://localhost:4000`.

---

## How to Use

Browse the catalog directly through the web interface at the project page. To add a new movie:

1. Create a new Markdown file in the `_movies` directory following the existing template format
2. Include title, year, description, and any available metadata
3. Submit a pull request with your addition

For local development, use the Jekyll server to preview changes before submitting.

---

## Configuration

All movie entries are stored as individual Markdown files in the `_movies` folder. Site-wide settings can be found in `_config.yml`. No additional database or server configuration is required.
